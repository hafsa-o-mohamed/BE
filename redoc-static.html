Tap Payments API (1.0.0)

Download OpenAPI specification:

API for processing payments through Tap Payments with Apple Pay integration.

This API allows clients to process Apple Pay payments and receive webhook notifications from Tap Payments.

Payments

Payment processing endpoints

Process Apple Pay payment

Process a payment using Apple Pay token.

This endpoint accepts an Apple Pay token along with payment details, converts it to a Tap Payments token, and creates a charge.
